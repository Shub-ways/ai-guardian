# AI Guardian 🚀

AI Guardian is a multi-modal AI-powered fraud detection system that identifies phishing links and scam messages using Machine Learning, rule-based security logic, and Google Gemini.

## 🔥 Features

* URL Phishing Detection (ML + Rules)
* Scam Message Detection (Gemini AI)
* Unified Risk Scoring System
* Explainable AI (Reasons + Breakdown)
* Confidence Scoring

## 🧠 Tech Stack

* FastAPI (Backend)
* Scikit-learn (ML)
* Google Gemini API (LLM)
* Python

## 🚀 API Endpoints

* `/analyze-url`
* `/analyze-text`
* `/analyze` (Unified)

## 🎯 Objective

To protect users from digital fraud by providing real-time, explainable risk analysis.
